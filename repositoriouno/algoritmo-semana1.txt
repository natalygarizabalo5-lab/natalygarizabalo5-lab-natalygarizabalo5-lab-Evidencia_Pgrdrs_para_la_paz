Nombre del algoritmo: Cómo compartir información confiable en internet.
Inicio
1. Me llegó la información
2. Leo la información
3. Investigo fuentes
4. Corroboro fuentes
5. Confirmo información 
6. Comparto información
Fin.

    // Paso 3: Investigar fuentes
    Buscar fuentes relacionadas con informacion
 
    // Paso 4: Corroborar fuentes
    Comparar fuentes encontradas
    Si (fuentes son consistentes) Entonces

    // Paso 5: Confirmar información

    FinSi
 
    // Paso 6: Compartir información
    Si (validacion = Verdadero) Entonces
        Compartir informacion
    Sino
        Mostrar "No se comparte la información"
    FinSi
